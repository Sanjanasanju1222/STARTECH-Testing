## 🛒  Star Tech Website-Testing

📌This repository contains Software Quality Assurance (SQA) testing documentation for the Star Tech e-commerce website. It includes test plan, test cases, execution reports, bug reports, mind maps, and screenshots. The project showcases practical experience in manual testing, defect reporting, test case design, and QA documentation.
